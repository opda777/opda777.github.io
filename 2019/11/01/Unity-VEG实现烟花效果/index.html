前言

主粒子

设置主粒子的速度，位置，周期等参数。

Position(Line)

用于设置粒子的初始化位置，从Start到End随机生成一个位置点。

烟花粒子

这里利用GPU Event当主粒子消亡时生成新的眼花粒子。

GPU Event
